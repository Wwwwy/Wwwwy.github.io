2017-02-09 15:20:00

1. MAP과 LIST

쿼리의 결과나 데이터를 조작하기 위해서 자주 사용하는게 MAP과 LIST 입니다.
MAP은 Key와 Value의 묶음으로써 아래와 같은 데이터의 형태를 보입니다. ( 보기 편하기 위해 json 형태의 데이터로 표현했습니다. )

{ “data” : “데이터” }

Map을 선언할때 제너릭스를 통해서 명확한 타입선언이 가능한데 이 경우 value값을 Object 또는 String으로 선언 후 원하는대로 형변환을 통해 사용이 가능합니다.

List는 배열의 약점을 보완하기 위해서 만들어진 자바의 내장 자료형입니다.
